## Key Insight

The most impactful terms were NOT the ones predicted by theorists. They emerged from OBSERVING git-agents evolving in the wild. The terms that stuck were:
1. PRACTICAL (usable in everyday dev work)
2. COMPRESSIBLE (one word replaces a paragraph)
3. BRIDGEABLE (connects across domains)
4. EXECUTABLE (can be compiled into agent behavior)
